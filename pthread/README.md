线程时程序执行的最小单位
进程时资源竞争的最小单位

线程自己拥有：
    线程id
    程序计数器
    寄存器组
    栈
    errno
与进程共享：
    代码段
    数据段
    打开的文件句柄和信号

易错点：
    给每个线程传递的都是主线程中变量的地址，
    如果主线程中修改了这个变量，将影像线程中取这个变量的值
